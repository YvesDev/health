⚡ 饮食原则

主食控量 每餐 100g 以内（生重），优先糙米 / 燕麦 / 红薯 / 玉米，少吃白米饭
蛋白质优先 每餐必须有，鸡胸 / 鱼 / 虾 / 蛋 / 豆腐轮换搭配
低脂烹饪 蒸、煮、少油炒为主，每天用油不超过 25g
零食 & 饮水 饿了吃黄瓜 / 番茄 / 少量坚果，不喝含糖饮料，每天≥1500ml 水

💡 关键贴士
🌾
主食选择

每餐 100g 以内（生重），糙米 / 燕麦 / 红薯 / 玉米优先，白米饭尽量少吃

🍗
蛋白质补充

每餐都要有蛋白质来源，鸡胸 / 鱼 / 虾 / 蛋 / 豆腐轮换，保持饱腹感

🥘
低脂烹饪

以蒸、煮、少油炒为主，每天用油总量不超过 25g

🥒
零食管理

饿了可以吃黄瓜、番茄、少量坚果，绝对不喝含糖饮料

💧
充足饮水

每天至少 1500ml 水，有助于提高代谢、减少饥饿感

🚶 久坐提醒：哪怕每小时起来走动 5 分钟，也对减脂有明显帮助，坚持下去效果更好！
